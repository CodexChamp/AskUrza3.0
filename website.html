What's Ask Urza 3.0?

Ask Urza 3.0 is a fan-made, reimagined version of Ask Urza 2.0 which is a fan-made, reimagined version of the "Ask Urza" webpage made by Wizards of the Coast for a card named Urza, Academy Headmaster. Ask Urza traditionally has 20 randomized effects for each of its three loyalty abilities, but Ask Urza 2.0 takes things a step further. That was created by Kyle Burton. So what makes mine different beside better art? This one isnt balanced. I have taken any Loyalty abilty that takes more than -4 to activate and added that as an ultimate. So if there is an ultimate you like, he has it. I have also taken some leeway and created a few abilities that some could be entertained by. So have fun.
